# Assembly Guide | 組裝指南

**Warning**: This system involves high-pressure piping, electrical systems, combustion devices, and food-grade materials. Assembly should only be attempted by individuals with relevant technical knowledge and safety awareness.

### Assembly Overview
1. Assemble the main tower structure (Layers 1–4)
2. Install the external Water-Energy Recovery Module
3. Connect all piping and electrical wiring
4. Install sensors and control system
5. Fill with culture medium and inoculate microalgae
6. Perform system testing and calibration

**Detailed step-by-step instructions will be added in future updates.**  
Estimated assembly time: 2–4 weeks depending on experience.

**Important**: All food-contact surfaces must use food-grade materials and be thoroughly sanitized before first use.
